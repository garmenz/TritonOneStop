# TritonOneStop_Frontend
installation guide for backend
1. install PyCharm using NetJetBeans account
2. try to run app.py
3. In PyCharm, Setting => Project Intepreter => choose python3.7 on the top => + => add required package
4. Dependencies: 
flask
firebase_admin
flask_cors
firebase
python_jwt
gcloud
sseclient
pycrypto
requests_toolbelt
